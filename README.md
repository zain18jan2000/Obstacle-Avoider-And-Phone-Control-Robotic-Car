# Obstacle-Avoiding Car

<h1>COMPONENTS USED</h1><br><br>

1- Arduino uno <br>
2- Moter driver L298N <br>
3- BO Motor x 2 <br>
4- Centre wheel <br>
5- Servo Motor <br>
6- Ultrasonic Sensor HC-SRO4 <br>
7- 12v Battery  <br>

9- Jumper wires <br>
